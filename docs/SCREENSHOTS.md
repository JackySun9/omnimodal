## ğŸ“ How to Contribute Screenshots

If you'd like to contribute screenshots or demos:

1. **Take screenshots** of the application in action
2. **Create GIFs** using tools like:
   - [LICEcap](https://www.cockos.com/licecap/) (Windows/macOS)
   - [Kap](https://getkap.co/) (macOS)
   - [Peek](https://github.com/phw/peek) (Linux)
3. **Optimize images**:
   - PNG for static screenshots (compress with [TinyPNG](https://tinypng.com/))
   - GIF for animations (keep under 5MB)
   - Use 1920x1080 or 1280x720 resolution
4. **Save to** `docs/images/` directory
5. **Update this file** with descriptions
6. **Submit a PR** with your additions
